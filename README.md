Study Buddy
The Study Buddy is a mobile application productivity tool intended to enhance user organization. 
Its primary advantage lies in the direct integration of structured scheduling which is document management. 
The platform is fundamentally structured around two critical components: secure identity management,
personalized academic task planning. The initial development phase will prioritize robust user authentication,
core scheduling functionality.
Its Features are:
User Registration: Allows new users to establish an account within the system 
it requires the users email address, password, Username, and name.
User Authentication: Facilitates secure login for returning users along with standardized
email/password validation and reliable session management are essential to maintain user persistence.
Profile Modification: Enables users to update personal and academic data following initial registration and its capabilities include updating user details.
Data Persistence: Management of all user and profile data must be secure, leveraging a scalable database solution, 
Firebase is specified for secure, private data storage under the unique user ID.
